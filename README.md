# Bootstrap 5 UI Internship Task

## Project Overview
This project is developed as part of an internship task to explore and apply Bootstrap 5 UI components by remixing them into clean, responsive web pages.

## Step-by-Step Process
1. Studied Bootstrap 5 official documentation and examples
2. Selected reusable components like navbar, cards, forms, and modals
3. Designed three pages: Home, About, and Contact
4. Ensured responsive layout using Bootstrap grid and utilities
5. Tested the UI on different screen sizes
6. Pushed the project to GitHub and deployed it

## Tools & Resources Used
- Bootstrap 5 Documentation
- Bootstrap Examples
- ChatGPT (for guidance and code structuring)
- GitHub for version control and hosting

## Challenges Faced
- Maintaining consistent spacing and alignment
- Understanding responsive behavior on smaller screens  
These were solved by using Bootstrap utility classes and grid system.

## Learning Outcome
- Hands-on experience with Bootstrap 5
- Improved UI/UX composition skills
- Learned GitHub workflow and deployment
